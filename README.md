# 中文语音识别系统 - Vosk + 置信度校准网络

> 基于 Vosk 离线语音识别引擎与自训练置信度校准网络的中文语音识别系统。  
> 使用 THCHS-30 数据集训练一个二分类器，自动判断 Vosk 识别结果是否需要人工纠错。  
> 课程设计作品 | 仅供技术交流参考

## ✨ 功能

- 🎙️ **离线中文识别**：调用 Vosk 小型中文模型，无需联网即可将语音转为文字。
- 🧠 **置信度校准网络**：训练一个轻量级神经网络，根据 Vosk 置信度、文本长度等特征，预测识别结果是否需要纠错。
- 📊 **自动评估**：在 THCHS-30 数据集上自动生成训练/验证集，评估校准模型准确率。
- 📈 **训练可视化**：实时绘制损失和准确率曲线，直观观察训练过程。
- 💾 **模型持久化**：训练完成后自动保存校准网络权重和字符词表，便于后续加载使用。

- Python 版本：`3.7` ~ `3.10`（需兼容 TensorFlow）
- 硬件：仅需 CPU，神经网络规模很小
